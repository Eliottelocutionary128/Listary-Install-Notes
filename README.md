## 🛑 Common install errors

Many users report the installation failing mid-way through. Use these fixes if you reach a roadblock:

### Setup file appears as corrupted
If the installer reports a corrupted file, your browser likely blocked a partial download. Clear your browser cache, restart your internet connection, and attempt the download one more time.

### Error message about existing processes
If you see an error saying a file is in use, verify that no previous versions of the software remain running. Open your Task Manager, find any process starting with "Listary," and right-click to "End Task." Then, restart the installation.

### Access Denied warnings
Windows 11 features strict permission controls. If the setup fails with "Access Denied," ensure you are running the installer with administrative privileges. If you are on a work computer, contact your system administrator to ensure your profile allows the installation of search indexing tools.

### Software does not launch after install
If the app installs correctly but refuses to open, verify that your firewall is not blocking the application. Occasionally, deep-system scanners mark search tools as threats. Add an exception for the installation folder in your Windows Security preferences.
